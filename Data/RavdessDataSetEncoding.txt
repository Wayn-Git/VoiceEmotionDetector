Ravdess Dataset has the metadata directly in the file name for examples ->

03-01-05-01-02-02-02-12.wav

each part has meaning actor, emotion, statement, repetition etc

For ravdess specifically: 

MM-SS-EE-II-RR-VV-AA.wav

MM -> Speech, Song
SS -> Vocal Channel
EE -> Emotion (01 = neutral, 02 = calm, 03 = happy, 04 = sad, 05 = angry, 06 = fearful, 07 = disgust, 08 = surprised)
II -> Intensity (Normal/Strong)
RR -> statement (1 or 2)
VV → Repetition (1 or 2)
AA → Actor ID
